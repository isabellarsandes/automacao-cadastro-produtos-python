# 🤖 Automação de Cadastro de Produtos com Python

## 📌 Sobre o projeto

Este projeto foi desenvolvido durante a Jornada Python, com o objetivo de automatizar tarefas repetitivas utilizando a linguagem Python.

A solução simula a atuação de um usuário no computador, realizando o cadastro automático de produtos em um sistema web a partir de uma base de dados em formato CSV.

---

## 🎯 Problema de negócio

Processos operacionais que envolvem cadastro manual de grande volume de informações podem gerar:

- perda de produtividade  
- maior risco de erros  
- alto tempo de execução  

Neste contexto, o projeto demonstra como a automação pode ser utilizada para aumentar a eficiência operacional e reduzir falhas humanas.

---

## 📊 Base de dados

- Arquivo CSV contendo:
  - código do produto  
  - marca  
  - tipo  
  - categoria  
  - preço  
  - custo  
  - observações  

A automação percorre todas as linhas da base e realiza o cadastro automático no sistema.

---

## ⚙️ Funcionamento da automação

O script executa as seguintes etapas:

- abertura automática do navegador  
- acesso ao sistema com login  
- preenchimento dos campos do formulário  
- envio das informações  
- repetição do processo até finalizar todos os produtos  

---
